# Youtube-View-Predictor-App
an app for predicting views for any youtube video using historical data and features of the video.

# overview
machine learning model to predict youtube video views based on historical video metadata and engagement features.

# features
- data preprocessing and exploratory data analysis
- regression-based prediction models
- performance evaluation

# tech stack
- python
- pandas
- numPy
- scikit-learn

# use case
shows practical application of machine learning through prediction of views and vizualization with data analytics concepts.

It is and Academic project.
